2.1 查询某段日期单支股票数据

ID 1 名称 查询某段日期单支股票数据
创建者 黄泽鹏 最后一次更新者  
创建日期 2017/3/4 最后更新日期  
参与者 用户，目标是快速了解单支股票最近数据的涨跌情况
触发条件 用户想要了解某支股票数据的涨跌情况
前置条件 用户已登录
后置条件 无
优先级 中
正常流程
  1. 用户输入开始日期、结束日期和股票编号，点击搜索
  2. 系统显示该时间段内该股票的单日开盘价、收盘价、最高价、最低价
扩展流程 
 1a:日期不在范围内
   1.系统提示日期不在有效范围内，并要求重新输入
 1b:日期跨度过大
   1.系统提示日期跨度过大，并提示日期跨度限制
 1c:股票编号无效
   1.系统提示股票编号无效，并要求重新输入
特殊需求
  1. 日期范围是2005年2月到2014年4月。
  2. 日期跨度是1年。

2.3 股票比较

ID 3 名称 股票比较
创建者 刘宇翔 最后一次更新者 刘宇翔
创建日期 2017年3月4日 最后更新日期 2017年3月4日
参与者 用户，目标是查看两只股票的对比信息。
触发条件 用户将两只股票编号或股票名字添加到比较栏
前置条件 用户知道股票的编号或名字
后置条件 展示相关股票信息
优先级 高
正常流程
  1. 用户选择股票比较
  2. 系统给出股票名字或股票编号的输入框以及查询起始时间与终止时间的输入框。
  3. 用户输入需要对比的股票名字或股票编号以及查询的起始时间与终止时间。
  4. 系统显示两只对比股票的所有从起始时间到终止时间的最低值、最高值、涨幅/跌幅、每天的收盘价和对数收益率、对数收益率方差。
  5. 用户选择返回.
  6. 系统显示起始界面
扩展流程 
3a. 用户输入错误的股票名字或股票编号
	1.系统提示输入了错误的股票名字或股票编号
	2.回到正常流程2
3b. 用户输入错误的日期范围有误
	1.系统提示错误的日期范围
	2.回到正常流程2
特殊需求
  1. 日期范围是2005年2月到2014年4月。

3.功能需求

3.1 查看某段日期单支股票数据和走势

3.1.1 特性描述

用户有查看某支股票一段时间内的涨跌情况的意愿，并输入了开始结束时间、股票代码进行查询，系统应显示股票数据，在用户选择查看图表时进行相关图表显示。

优先级=中。

3.1.2 刺激/响应序列

刺激：客户输入的日期范围超出有效范围
响应：系统提示日期无效，删除已输入日期，并要求重新输入

刺激：客户输入的日期范围跨度过大
响应：系统提示日期跨度过大，删除已输入日期，并要求重新输入

刺激：客户输入的股票代码不存在
响应：系统提示股票代码不存在，删除已输入股票代码，并要求重新输入

刺激：客户输入正确股票信息并点击搜索
响应：系统显示该段日期内该股票的数据

刺激：客户选择查看Ｋ线图（均线图）
响应：系统显示该段日期内该股票的Ｋ线图（均线图）

3.1.3 相关功能需求

Stock.Search.Input 系统应允许客户在搜索股票信息任务中进行键盘输入
Stock.Search.Input.Calendar 在用户输入日期范围时，系统显示日期日历
Stock.Search.Input.Info 在用户请求查看股票详细信息时，系统要查询相应股票信息并显示， 参见Stock.Search.Info
Stock.Search.Input.Invaild 用户输入日期超出范围、跨度过大或者股票代码不存在，系统提示对应错误，参见Stock.Search.Invaild
Stock.Search.Input.Del 用户输入无效时，系统删除已输入内容
Stock.Search.Info 系统显示该股票的数据
Stock.Search.Info.Data 用户请求查看该股票的数据，系统显示该股票的数据
Stock.Search.Info.KandEMA 用户请求查看该股票的K线图和均线图，系统显示该股票的K线图和均线图
Stock.Search.Invaild 系统提示输入无效
Stock.Search.Invaild.Calendar 系统提示输入日期不在有效范围内
Stock.Search.Invaild.Range 系统提示输入日期跨度过大
Stock.Search.Invaild.Code 系统提示输入股票代码不存在

3.2 股票比较

3.2.1 特性描述

当用户希望去对比所选中的股票在用户选中的时间范围内的情况时，系统需要提供给用户所选的两只股票在用户选定时间范围内的所有数据信息， 并且显示两只对比股票的所有从起始时间到终止时间的最低值、最高值、涨幅/跌幅、每天的收盘价和对数收益率、对数收益率方差。

优先级=高

3.2.2 刺激响应序列

刺激：用户输入正确的股票编号或是股票名字以及时间范围。

响应：系统显示对应的股票的所有从起始时间到终止时间的最低值、最高值、涨幅/跌幅、每天的收盘价和对数收益率、对数收益率方差。

刺激：用户选择退出股票对比功能。

响应：系统显示股票软件初始界面。

刺激：用户输入了错误的股票编号或名字。

响应：系统提示输入的股票编号或名字有误。

刺激：用户输入了错误的时间范围。

响应：系统提示错误的时间范围。

3.2.3 相关功能需求

Shares_Compare.Input
Shares_Compare.Input.Shares
Shares_Compare.Input.Date		 系统应该允许用户在对比股票数据时进行键盘输入。
系统需要接收用户输入的股票编号或股票名字。
系统需要接收用户输入的时间。
Shares_Compare.Judge
Shares_Compare.Judge.Shares
Shares_Compare.Judge.Date		 系统会对于用户的输入进行判断。
系统判断所接收的股票编号或名字是否正确，如不正确，跳转到.Error.Shares。
系统判断所接收的时间是否正确,如不正确，跳转到.Error.Shares。
Shares_Compare.Error
Shares_Compare.Error.Shares
Shares_Compare.Error.Date		 系统对于错误的操作进行提示。
系统提示输入的股票编号或名字有误。
系统提示错误的时间范围。
Shares_Compare.Compute
Shares_Compare.Compute.Info		 系统需要对于获取的股票数据进行处理。
系统根据获取的股票数据计算所有从起始时间到终止时间的最低值、最高值、涨幅/跌幅、每天的收盘价和对数收益率、对数收益率方差。
Shares_Compare.Output 系统显示两只股票的所有从起始时间到终止时间的最低值、最高值、涨幅/跌幅、每天的收盘价和对数收益率、对数收益率方差的数据。
Shares_Compare.Exit 系统显示股票软件初始界面。




3.3 查看股票比较图

3.3.1 特性描述

当用户希望去对比所选中的股票的k线图与均线图的对比情况时，系统需要提供给用户所选的两只股票的k线图与均线图的绘制与对比。

优先级=高

3.3.2 刺激响应序列

刺激：用户选择查看对比股票的k线图/均线图。

响应：系统显示对应的股票的k线图与均线图。

刺激：用户选择返回股票信息对比。

响应：系统显示股票的信息数据对比界面。

3.3.3 相关功能需求

Draw_Compare.GetData 系统获取两只对比股票的所有数据。
Draw_Compare.Output
Draw_Compare.Output.kchat
Draw_Compare.Output.avgchat		 系统绘制出两只对比股票的相关对比图。
系统绘制出两只对比股票的k线图。
系统绘制出两只对比股票的均线图。
Draw_Compare.Exit 系统显示股票的信息数据对比界面。

3.4 查看市场温度计

3.4.1 特性描述

当用户希望去获取特定日期整个股票交易市场的情况时，系统需要提供给用户所选的日期的股票市场信息， 并且显示当日总交易量、涨停股票数、跌停股票数、涨幅超过 5%的股票数，跌幅超过 5%的股票数，开盘‐收盘大于 5%*上一个交易日收盘价的 股票个数、开盘‐收盘小于‐5%*上一个交易日收盘价的股票个数。

优先级=高

3.4.2 刺激响应序列

刺激：用户输入正确的日期。

响应：系统显示对应的日期的股票市场的统计数据。

刺激：用户选择退出股票对比功能。

响应：系统显示股票软件初始界面。

刺激：用户输入了错误的时间。

响应：系统提示错误的时间。

3.4.3 相关功能需求

Market_Info.Input.Date

 系统应该允许用户在获取市场温度计情况时进行键盘输入。

Market_Info.Judge
Market_Info.Date		 系统会对于用户的输入进行判断。

Market_Info.Error
Market_Info.Error.Date		 系统对于错误的操作进行提示。
系统提示错误的时间范围。
Market_Info.Compute

系统根据获取的股票数据计算当天总交易量、涨停股票数、跌停股票数、涨幅超过 5%的股票数，跌幅超过 5%的股票数，开盘‐收盘大于 5%*上一个交易日收盘价的 股票个数、开盘‐收盘小于‐5%*上一个交易日收盘价的股票个数。
