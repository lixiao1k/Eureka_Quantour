1.用例列表

1.1 查看市场温度计

ID 1 名称 查看市场温度计
创建者 李晓冬 最后一次更新者  
创建日期 2017/3/21 最后更新日期  
参与者 用户，目标是快速了解某市场行情
触发条件 用户想要了解某市场基本行情
前置条件 用户已登录
后置条件 无
优先级 高
正常流程
  1. 用户输入查看的某一日期并选择市场，点击搜索
  2. 系统显示该天市场的市场温度计信息、股票涨跌幅排名、大盘指数、所有股票信息列表
  3. 市场温度计信息包括：当日总交易量、涨停股票数、跌停股票数、涨幅超过5%的股票数，跌幅超过5%的股票数，开盘‐收盘大于5%*上一个交易日收盘价的股票个数、开盘‐收盘小于‐5%*上一个交易日收盘价的股票个数。
  4. 股票涨跌幅排名只需各显示10条数据，每条数据包括编号、名称、涨跌额、涨跌幅
  5. 大盘指数只当天的大盘指数
  6. 股票信息列表中每条信息包括序号、代码、名称、最新价、涨跌幅、涨跌额、开收盘价、最高低价、交易量
扩展流程
  1. 1.系统提示日期不在有效范围内，并要求重新输入
  2. 2.用户在浏览股票信息时，可以查看单只股票的基本信息，包括涨跌幅、涨跌额、开收盘价、最高低价、交易量、K线均线图。
特殊需求
  1. 日期显示
  2. K线均线图的范围根据选择的日期向前推适当天数


2.用户制定策略

ID   名称 用户制定策略
创建者 惠晗涛 最后一次更新者  惠晗涛
创建日期 2017/3/23 最后更新日期 2017/3/23 
参与者 用户，目标是使用户制定筛选股票的策略，从而提高收益率
触发条件 用户选择制定策略
前置条件 用户已完成股票池的选择
后置条件 显示策略所对应的图像与参数，来预估策略的收益率与风险程度
优先级 高
正常流程
  1. 用户设置策略相关参数，股票数量，调仓时间，调仓价格，股票池，数据开始时间，删选条件，删选数量
  2. 用户选择生成策略
  3. 系统显示策略所对应的相关图像，包括收益率图，超额收益率与侧略胜率图像
  4. 系统显示策略的相关参数，阿尔法，贝塔，最大回撤率，收益波动率
扩展流程 
1a:用户设置股票数量大于股票池数量
1.弹出错误框，并要求用户重新输入
1b:调仓时间不超过100天
1.弹出错误框，并要求用户重新输入
1c：数据开始时间距离现在时间小于一年
1.弹出错误框，并要求用户重新输入
特殊需求

1.调仓价格限定在开盘价，收盘价，最高价于最低价算术平均值

2.筛选条件至少包括动量策略（形成期内收益率的排序），与均值策略（调仓价于均线图偏离率的排序）。

用例4 用户管理

ID 1 名称 用户管理
创建者 刘宇翔 最后一次更新者 刘宇翔
创建日期 2017年3月21日 最后更新日期 2017年3月21日
参与者 用户，目标是注册该软件的一个账户，从而实现对于股票的收藏关注功能。
触发条件 用户在登录界面点击登录或注册
前置条件 用户的用户名与密码符合规定由数字和字母组成且大于6位并且少于10位。
后置条件 无
优先级 中
正常流程
  1. 用户选择注册
  2. 系统显示用户名，密码，重复密码的输入框。
  3. 用户输入用户名，密码，重复密码。
  4. 系统显示注册成功并进入股市行情界面。
扩展流程 
1a. 用户选择登录
	1.用户输入用户名与密码
	2.系统显示登录成功，进入股市行情界面
	2a.用户输入错误的用户名或密码
		1.系统显示用户名或密码不正确
4a. 用户输入的用户名或密码不是由数字和字母组成并且大于6位并小于10位
	1.系统提示输入了不合法的用户名或密码
	2.回到正常流程2
4b. 用户输入的重复密码与密码不一致
	1.系统提示输入了不一致的重复密码与密码
	2.回到正常流程2
4c. 用户选择登出
	1.系统登出该用户账号
特殊需求
  1. 用户的用户名与密码符合规定由数字和字母组成且大于6位并且少于10位。
